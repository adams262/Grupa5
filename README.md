Aplikacja generująca wniosi/podsumowanie po wklejeniu linku z yt



Funkcjonalne: 
* Wprowadzanie URL: System musi umożliwiać użytkownikowi wklejenie linku do filmu z YouTube
* Pobieranie danych i pobieranie transkrypcji: System musi przetworzyć dane wejściowe
* Konwersja Audio to Text: System musi przetworzyć dźwięk na tekst
* Podsumowanie: System powinien generować wnioski na podstawie zadanego materiału wideo
* Wyświetlanie wyników: System powinien wyświetlać wygenerowany tekst



Niefunkcjonalne:
* Dostępność: Aplikacja musi poprawnie wyświetlać się na ekranach urządzeń mobilnych.
* Wydajność: Aplikacja powinna działać płynnie oraz relatywnie szybko (optymalizacja).
* Skalowalność: Aplikacja powinna obsłużyć aparat ze smartfonu bez awarii.
* Bezpieczeństwo: Aplikacja powinna używać tylko wymaganych uprawnień urządzenia.
